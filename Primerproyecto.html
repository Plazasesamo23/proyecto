Github

Instal·lació de Github

Primer de tot vam començar instal·lant-nos el Github

Un cop instal·lat, vam baixar-nos el git , on seguidament ens vam registrar. Aquestes van ser les comandes que vam utilitzar:
Primer per a instal·lar el Git vam fer servir la següent comanda:
apt install git
Després vam comprovar la versió instal·lada:
ism47955722@a10:~$ git version
Després vam posar el nostre nom d’usuari:
ism47955722@a10:~$ git config --global user.name "Plazasesamo23”
Més tard vam establir l’email:
ism47955722@a10:~$ git config --global user.email "bryanarevalo1912@gmail.com"

Per inicialitzar el repositori que volem per al nostre projecte:
git init "Nom de la carpeta"
Després per a agregar documents o arxius fem el següent:
git add "Nom dels documents"
I per a acabar de pujar l'arxiu amb un missatge (Això és obligatori):
git commit -m “missatge”

Configuració de les branques:
(Creació d'una branca)
git branch "Nom de la nova branca"
Vam pasar a utilitzar aquesta branca com a principal:
git checkout -b "Nom de la branca"
Amb aquesta comanda seleccionem la branca amb la qual més tard pujarem tots els arxius:
git push origin "Nom de la branca"

I per a baixar els documents al servidor un git
git pull
